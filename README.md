📌 Overview

A console-based Bank Account Management System developed in C that uses random-access file handling to store and manage account records efficiently.

This project demonstrates core concepts of:

Structured programming

Binary file operations

Persistent data storage



✨ Features

➕ Create new account

📄 Display all accounts

🔍 Search account by ID

✏️ Update account details

❌ Delete account

💾 Data stored in file (accounts.txt)
